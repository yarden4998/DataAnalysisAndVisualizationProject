## Set up code environment
1. Make sure to have poetry installed on your computer. 
If not, please install by running:
    ```bash
    pip install poetry
    ```
2. Clone this repo.
3. Run from main root:
    ```bash
    poetry install
    ```
Now everything is set and you are ready to run the code.
